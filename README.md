# RedThread-DataAnalyticsProject
 Projet de Fin d’Études – Analyse Prédictive &amp; Segmentation Clientèle

## 📌 Description
Ce projet vise à exploiter des jeux de données réels liés à la fidélisation client et aux comportements transactionnels afin de construire des modèles analytiques et prédictifs. L'objectif est d'aider les entreprises à mieux comprendre leurs clients, anticiper les risques de churn (désabonnement), optimiser leurs offres marketing et améliorer leur prise de décision stratégique.

## 🎯 Objectifs Principaux
1. Identifier les facteurs clés influençant le churn client.
2. Segmenter la clientèle selon des critères géodémographiques et comportementaux.
3. Prédire la valeur future des clients via des modèles de régression.
4. Évaluer la performance des modèles à l’aide de métriques pertinentes (CAP Curve, AUC).
5. Automatiser un pipeline ETL pour assurer la reproductibilité.
